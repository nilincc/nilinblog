记录一下在GitHub上传代码与文件的过程
发布于：2023-09-19 22:19:17 标签：/ GitHub / 访问：

今天在网上闲逛的时候发现了一个非常喜欢的一个静态的页面，想着自己的nilin.cc的首页也很久没有换过了，想着换一下，前面使用git命令还是很nice的，结果现在的分支改成了main这个就感觉有点坑了！结果一直出现错误！

正好也记录一下这个操作过程。

git init //把这个目录变成Git可以管理的仓库

git add README.md //文件添加到仓库

git add . //不但可以跟单一文件，还可以跟通配符，更可以跟目录。一个点就把当前目录下所有未追踪的文件全部add了

git commit -m “first commit” //把文件提交到仓库

git remote add origin git@github.com:nilincc/nilincc.github.io.git //关联远程仓库

git push -u origin master //把本地库的所有内容推送到远程库上

以上的这些命令应该会玩git的大佬应该都不陌生吧，但是今晚上遇到了一个比较辣手的问题，
